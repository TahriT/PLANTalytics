About PLANTalytics

Understanding plant data for sustainable agriculture and permaculture

Project Overview

PLANTalytics is a comprehensive data analytics project designed to help gardeners, farmers, and permaculture enthusiasts better understand plant growth patterns, synergies, and characteristics. By aggregating data from trusted sources like Plants For A Future (PFAF) and PermaPeople, we provide actionable insights for sustainable growing practices.

Data Sources

  • PFAF.org: Comprehensive database of edible and useful plants
  • PermaPeople.org: Permaculture-focused plant database with ecological relationships
  • USDA Hardiness Zones: Climate and geographical data integration

Features

  • Interactive Power BI dashboards with real-time filtering
  • Comprehensive plant characteristic analysis (20+ attributes)
  • Companion planting and ecosystem service insights
  • Climate zone and habitat range mapping
  • Edibility and medicinal use ratings
